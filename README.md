# wk-frontend-design

打造具有高设计质量的独特生产级前端界面。

当需要构建网页组件、页面、应用程序或任何界面时使用此技能。生成富有创意、精致的代码和界面设计，避免千篇一律的 AI 美学。

## 设计理念

**选择清晰的概念方向并精准执行。** 大胆的繁复主义和精致的极简主义同样有效——关键在于意图感，而非强度。

## 核心原则

### 设计流程

1. **理解背景** — 界面解决什么问题？谁会用？
2. **确定基调** — 极简、繁复、复古未来、有机自然、奢华精致、俏皮玩具、时尚杂志、野兽派、装饰艺术、柔和粉彩、工业实用...
3. **明确限制** — 技术要求（框架、性能、可访问性）
4. **差异化** — 是什么让人难忘？用户会记住的那个点是什么？

### 严禁行为

禁止使用通用的 AI 生成审美：

| 类别 | 禁止模式 | 替代方案 |
|------|----------|----------|
| 字体 | Inter/Roboto/Arial/system-ui 作为展示字体 | Playfair Display、Cormorant Garamond、Bebas Neue 等 |
| 配色 | 白色背景 + 紫色渐变 | 60-30-10 法则，锐利对比 |
| 布局 | 完全对称的 Flex 均匀分布 | 不对称、重叠、打破网格 |
| 动画 | 单一元素简单 hover 缩放 | 交错编排的进入/退出动画 |
| 阴影 | 平庸的 `box-shadow: 0 2px 5px rgba(0,0,0,0.1)` | 戏剧性阴影、渐变光晕 |

## 字体推荐

### 展示字体（标题）

- **Playfair Display** — 经典衬线，优雅奢华
- **Cormorant Garamond** — 纤细衬线，文学气质
- **Bebas Neue** — 粗壮无衬线，力量感
- **Abril Fatface** — 装饰性衬线，戏剧张力
- **Syne** — 前卫几何，独特个性
- **Clash Display** — 超大显示，极端视觉

### 正文字体（正文）

- **Source Serif 4** — 现代衬线，屏幕优化
- **Libre Baskerville** — 经典衬线，权威感
- **DM Sans** — 几何无衬线，干净利落
- **Plus Jakarta Sans** — 当代几何，功能性

### 等宽字体（代码）

- **JetBrains Mono** — 编程首选
- **Fira Code** — 连字支持
- **IBM Plex Mono** — 工业美学

## 配色方法

### 60-30-10 法则

- **主色（60%）** — 背景和大面积色块
- **辅助色（30%）** — 次要元素、卡片、侧边栏
- **强调色（10%）** — 按钮、链接、关键数据

### 配色策略

| 策略 | 适用场景 | 示例 |
|------|----------|------|
| 单色系 | 极简、专业、数据仪表盘 | 不同明度的蓝色 |
| 互补色 | 需要吸引注意力、活泼 | 蓝 + 橙 |
| 类比色 | 和谐、自然、品牌感 | 绿 + 青 + 蓝 |
| 冷暖对比 | 戏剧性、情感表达 | 暖橙 + 冷紫 |

## 动画规范

### 时长指南

| 类型 | 时长 |
|------|------|
| 微交互（hover、toggle） | 100-200ms |
| UI 反馈（按钮点击、菜单展开） | 200-300ms |
| 页面过渡、模态框 | 300-400ms |
| 滚动触发动画 | 400-600ms |
| 进入/退出动画 | 300-500ms |

### 缓动函数

```css
ease-out  /* 自然减速，适合大多数进入动画 */
ease-in   /* 自然加速，适合退出动画 */
ease-in-out /* 对称运动，适合模态框 */
cubic-bezier(0.34, 1.56, 0.64, 1) /* 弹性回弹，适合按钮点击 */
```

## 输出要求

### 技术栈

- React + CSS Modules / Tailwind（默认）
- Vue 3 + Composition API
- 纯 HTML/CSS/JS（简单需求）
- TypeScript（默认使用）

### 响应式断点

- Mobile: 320px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+

### 可访问性标准（WCAG 2.1 AA）

- 颜色对比度 ≥ 4.5:1（正文）、≥ 3:1（大文本）
- 所有交互元素可通过键盘访问
- 提供适当的 ARIA 标签

### 性能目标

- 首屏加载 < 3s（3G 网络）
- Lighthouse 性能分数 ≥ 90
